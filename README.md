# Foodie


Hi! We are Group 23 in CS 2340 B and this is our website to tackle the Atlanta Food Finder Project! We utilized Django and the Google Maps API to help Atlanta residents find new restaurants near them. Our website enables users to register and create a profile. Users can also filter restuarants by name, cuisine, ratings, distance, and more. They can do this by using an interactive Google Maps and click on restuarant markers. They can also save their favorite restaurants so users can quickly revist them.

Dependencies:
- Python 3
- Django
- Pillow

To get this project, clone this repository, run `python manage.py migrate`, and then run `python manage.py runserver`.
